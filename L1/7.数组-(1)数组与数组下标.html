在C++中，数组是一种用于存储相同类型数据的集合。数组中的每个元素通过下标（索引）访问，下标从 0 开始。以下是数组及其下标的相关概念和用法：

1. 数组的定义

  • 语法： 
    数据类型 数组名[数组大小];
  • 示例： 
    int numbers[5];  // 定义一个包含 5 个整数的数组

2. 数组的初始化

  • 静态初始化：在定义数组时直接赋值。 
    int numbers[5] = {1, 2, 3, 4, 5};  // 初始化数组
  • 部分初始化：只初始化部分元素，未初始化的元素默认为 0。 
    int numbers[5] = {1, 2};  // 数组内容：{1, 2, 0, 0, 0}
  • 自动推断大小：根据初始化值的数量自动确定数组大小。 
    int numbers[] = {1, 2, 3, 4, 5};  // 数组大小为 5

3. 数组与下标

  • 下标：用于访问数组中的元素，下标从 0 开始。
  • 语法： 
    数组名[下标]
  • 示例： 
    int numbers[5] = {10, 20, 30, 40, 50};
cout << numbers[0] << endl;  // 输出：10
cout << numbers[2] << endl;  // 输出：30

4. 数组的遍历

可以使用循环遍历数组中的所有元素。

示例 1：使用 for 循环

int numbers[5] = {10, 20, 30, 40, 50};
for (int i = 0; i < 5; i++) {
    cout << numbers[i] << " ";
}
// 输出：10 20 30 40 50

示例 2：使用范围 for 循环（C++11）

int numbers[5] = {10, 20, 30, 40, 50};
for (int num : numbers) {
    cout << num << " ";
}
// 输出：10 20 30 40 50

5. 多维数组

C++支持多维数组（如二维数组、三维数组等）。

二维数组的定义与初始化

  • 语法： 
    数据类型 数组名[行数][列数];
  • 示例： 
    int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

二维数组的访问

  • 语法： 
    数组名[行下标][列下标]
  • 示例： 
    cout << matrix[0][1] << endl;  // 输出：2
cout << matrix[1][2] << endl;  // 输出：6

二维数组的遍历

for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        cout << matrix[i][j] << " ";
    }
    cout << endl;
}
// 输出：
// 1 2 3
// 4 5 6

6. 数组的注意事项

  1. 下标越界：

    • 访问数组时，下标必须在 0 到 数组大小 - 1 的范围内，否则会导致未定义行为。
    • 示例： 
      int numbers[5] = {1, 2, 3, 4, 5};
cout << numbers[5] << endl;  // 错误：下标越界

  2. 数组大小固定：

    • 数组的大小在定义时必须确定，且不能动态改变。

  3. 数组名是常量指针：

    • 数组名表示数组的首地址，是一个常量指针。
    • 示例： 
      int numbers[5] = {1, 2, 3, 4, 5};
int* ptr = numbers;  // ptr 指向数组的首地址

7. 综合示例

以下是一个完整的C++程序，演示了数组的定义、初始化、访问和遍历：

#include <iostream>
using namespace std;

int main() {
    // 定义并初始化数组
    int numbers[5] = {10, 20, 30, 40, 50};

    // 访问数组元素
    cout << "First element: " << numbers[0] << endl;  // 输出：10
    cout << "Third element: " << numbers[2] << endl;  // 输出：30

    // 遍历数组
    cout << "Array elements: ";
    for (int i = 0; i < 5; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;  // 输出：10 20 30 40 50

    // 二维数组
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    // 遍历二维数组
    cout << "Matrix elements:" << endl;
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }
    // 输出：
    // 1 2 3
    // 4 5 6

    return 0;
}

8. 总结

概念 说明 示例
数组定义 存储相同类型数据的集合 int numbers[5];
数组初始化 定义时直接赋值 int numbers[5] = {1, 2, 3, 4, 5};
数组下标 从 0 开始，用于访问数组元素 numbers[0] 访问第一个元素
数组遍历 使用循环访问数组中的所有元素 for (int i = 0; i < 5; i++) { ... }
多维数组 支持二维、三维等多维数组 int matrix[2][3];

数组是C++中非常重要的数据结构，熟练掌握其用法可以高效地处理大量数据。
