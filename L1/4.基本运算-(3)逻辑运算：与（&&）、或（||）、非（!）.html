在C++中，逻辑运算用于对布尔值（true 或 false）进行操作，常用于组合多个条件。逻辑运算符包括 与（&&）、或（||） 和 非（!）。以下是它们的详细说明和示例：

1. 逻辑与运算符 (&&)

  • 作用：当且仅当两个操作数都为 true 时，结果为 true；否则为 false。

  • 真值表：

    A B A && B
    true true true
    true false false
    false true false
    false false false

  • 示例：

    bool a = true, b = false;
bool result = a && b;  // result = false

2. 逻辑或运算符 (||)

  • 作用：当至少有一个操作数为 true 时，结果为 true；否则为 false。

  • 真值表：
    | A | B | A || B |
    |-------|-------|--------|
    | true | true | true |
    | true | false | true |
    | false | true | true |
    | false | false | false |

  • 示例：

    bool a = true, b = false;
bool result = a || b;  // result = true

3. 逻辑非运算符 (!)

  • 作用：对操作数取反。如果操作数为 true，结果为 false；反之亦然。

  • 真值表：

    A !A
    true false
    false true

  • 示例：

    bool a = true;
bool result = !a;  // result = false

4. 综合示例

以下是一个完整的C++程序，演示了逻辑运算符的用法：

#include <iostream>
using namespace std;

int main() {
    bool a = true, b = false;

    // 逻辑与
    cout << "a && b: " << (a && b) << endl;  // 输出：0 (false)

    // 逻辑或
    cout << "a || b: " << (a || b) << endl;  // 输出：1 (true)

    // 逻辑非
    cout << "!a: " << !a << endl;  // 输出：0 (false)
    cout << "!b: " << !b << endl;  // 输出：1 (true)

    return 0;
}

5. 短路求值

C++中的逻辑运算符支持 短路求值（Short-circuit Evaluation）：

  • 对于 &&，如果左操作数为 false，则不会计算右操作数。
  • 对于 ||，如果左操作数为 true，则不会计算右操作数。

示例

int x = 10, y = 20;

// 短路求值：x > 20 为 false，不会计算 y < 30
if (x > 20 && y < 30) {
    cout << "This won't be executed." << endl;
}

// 短路求值：x < 20 为 true，不会计算 y > 30
if (x < 20 || y > 30) {
    cout << "This will be executed." << endl;
}

6. 注意事项

  1. 布尔值输出：

    • 在C++中，true 被输出为 1，false 被输出为 0。
    • 可以使用 boolalpha 来输出 true 或 false： 
      cout << boolalpha << (a && b) << endl;  // 输出：false

  2. 非布尔值的逻辑运算：

    • 在C++中，非零值被视为 true，零值被视为 false。
    • 示例： 
      int x = 10, y = 0;
if (x && !y) {
    cout << "x is true and y is false" << endl;
}
