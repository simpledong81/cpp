在C++中，位运算直接对整数的二进制位进行操作。位运算常用于底层编程、性能优化和某些特定算法中。以下是C++中常见的位运算符及其用法：

1. 位与运算符 (&)

  • 作用：对两个操作数的每一位进行逻辑与操作。
    • 如果两个对应位都为 1，则结果为 1；否则为 0。
  • 示例： 
    int a = 5;    // 二进制：0101
int b = 3;    // 二进制：0011
int result = a & b;  // 结果：0001 (1)
cout << result << endl;  // 输出：1

2. 位或运算符 (|)

  • 作用：对两个操作数的每一位进行逻辑或操作。
    • 如果两个对应位中至少有一个为 1，则结果为 1；否则为 0。
  • 示例： 
    int a = 5;    // 二进制：0101
int b = 3;    // 二进制：0011
int result = a | b;  // 结果：0111 (7)
cout << result << endl;  // 输出：7

3. 位非运算符 (~)

  • 作用：对操作数的每一位进行逻辑非操作（按位取反）。
    • 将 0 变为 1，将 1 变为 0。
  • 示例： 
    int a = 5;    // 二进制：0101
int result = ~a;  // 结果：1010 (补码表示，实际值为 -6)
cout << result << endl;  // 输出：-6

4. 位异或运算符 (^)

  • 作用：对两个操作数的每一位进行逻辑异或操作。
    • 如果两个对应位不同，则结果为 1；否则为 0。
  • 示例： 
    int a = 5;    // 二进制：0101
int b = 3;    // 二进制：0011
int result = a ^ b;  // 结果：0110 (6)
cout << result << endl;  // 输出：6

5. 左移运算符 (<<)

  • 作用：将操作数的二进制位向左移动指定的位数，右侧补 0。
    • 左移一位相当于乘以 2。
  • 示例： 
    int a = 5;    // 二进制：0101
int result = a << 1;  // 结果：1010 (10)
cout << result << endl;  // 输出：10

6. 右移运算符 (>>)

  • 作用：将操作数的二进制位向右移动指定的位数，左侧补 0（对于无符号数）或符号位（对于有符号数）。
    • 右移一位相当于除以 2。
  • 示例： 
    int a = 5;    // 二进制：0101
int result = a >> 1;  // 结果：0010 (2)
cout << result << endl;  // 输出：2

7. 综合示例

以下是一个完整的C++程序，演示了各种位运算符的用法：

#include <iostream>
using namespace std;

int main() {
    int a = 5;  // 二进制：0101
    int b = 3;  // 二进制：0011

    // 位与
    cout << "a & b: " << (a & b) << endl;  // 输出：1

    // 位或
    cout << "a | b: " << (a | b) << endl;  // 输出：7

    // 位非
    cout << "~a: " << (~a) << endl;  // 输出：-6

    // 位异或
    cout << "a ^ b: " << (a ^ b) << endl;  // 输出：6

    // 左移
    cout << "a << 1: " << (a << 1) << endl;  // 输出：10

    // 右移
    cout << "a >> 1: " << (a >> 1) << endl;  // 输出：2

    return 0;
}

8. 注意事项

  1. 操作数类型：

    • 位运算符只能用于整数类型（如 int、char、long 等）。
    • 不能用于浮点数类型。

  2. 符号位：

    • 对于有符号数，右移运算符 (>>) 会保留符号位（算术右移）。
    • 对于无符号数，右移运算符 (>>) 会在左侧补 0（逻辑右移）。

  3. 位移范围：

    • 左移或右移的位数不能超过操作数的位数，否则行为未定义。
    • 示例： 
      int a = 1;
int result = a << 33;  // 未定义行为（假设 int 是 32 位）

  4. 位运算的用途：

    • 位运算常用于以下场景：
      • 性能优化（如快速计算 2 的幂）。
      • 位掩码操作（如设置或清除特定位）。
      • 加密算法或哈希函数。
