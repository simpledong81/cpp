在C++中，循环语句用于重复执行某段代码，直到满足特定条件。常见的循环语句包括 for 语句、while 语句和 do-while 语句。以下是它们的详细说明和示例：

1. for 语句

for 语句用于在已知循环次数的情况下重复执行代码。

语法

for (初始化; 条件; 更新) {
    // 循环体
}
  • 初始化：在循环开始前执行一次，通常用于初始化循环变量。
  • 条件：每次循环开始前检查，如果为真则执行循环体。
  • 更新：每次循环结束后执行，通常用于更新循环变量。

示例

for (int i = 0; i < 5; i++) {
    cout << "i = " << i << endl;
}

输出：

i = 0
i = 1
i = 2
i = 3
i = 4

2. while 语句

while 语句用于在条件为真时重复执行代码。

语法

while (条件) {
    // 循环体
}
  • 条件：每次循环开始前检查，如果为真则执行循环体。

示例

int i = 0;
while (i < 5) {
    cout << "i = " << i << endl;
    i++;
}

输出：

i = 0
i = 1
i = 2
i = 3
i = 4

3. do-while 语句

do-while 语句与 while 语句类似，但它的循环体至少会执行一次，然后再检查条件。

语法

do {
    // 循环体
} while (条件);
  • 条件：每次循环结束后检查，如果为真则继续循环。

示例

int i = 0;
do {
    cout << "i = " << i << endl;
    i++;
} while (i < 5);

输出：

i = 0
i = 1
i = 2
i = 3
i = 4

4. 循环控制语句

在循环中，可以使用以下控制语句改变循环的执行流程：

  • break：立即终止循环。
  • continue：跳过当前迭代，直接进入下一次循环。

示例

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // 当 i 等于 5 时终止循环
    }
    if (i % 2 == 0) {
        continue;  // 跳过偶数
    }
    cout << "i = " << i << endl;
}

输出：

i = 1
i = 3

5. 嵌套循环

循环语句可以嵌套使用，以处理更复杂的逻辑。

示例

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        cout << "i = " << i << ", j = " << j << endl;
    }
}

输出：

i = 1, j = 1
i = 1, j = 2
i = 1, j = 3
i = 2, j = 1
i = 2, j = 2
i = 2, j = 3
i = 3, j = 1
i = 3, j = 2
i = 3, j = 3

总结

循环类型 特点 适用场景 示例
for 已知循环次数 遍历数组、固定次数循环 for (int i = 0; i < 5; i++) { ... }
while 条件为真时重复执行 不确定循环次数 while (i < 5) { ... }
do-while 至少执行一次，条件为真时重复执行 需要至少执行一次 do { ... } while (i < 5);

这些循环语句是C++编程中实现重复操作的核心工具，熟练掌握它们可以编写出高效的程序。
