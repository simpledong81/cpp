在C++中，自增（++） 和 自减（--） 运算符用于对变量的值进行加 1 或减 1 操作。它们可以放在变量的前面（前缀形式）或后面（后缀形式），具体行为有所不同。以下是它们的详细说明和示例：

1. 自增运算符 (++)

  • 作用：将变量的值加 1。
  • 前缀形式：++变量
    • 先对变量加 1，然后返回加 1 后的值。
  • 后缀形式：变量++
    • 先返回变量的当前值，然后对变量加 1。

示例

int a = 5;

// 前缀形式
int b = ++a;  // a = 6, b = 6

// 后缀形式
int c = a++;  // a = 7, c = 6

2. 自减运算符 (--)

  • 作用：将变量的值减 1。
  • 前缀形式：--变量
    • 先对变量减 1，然后返回减 1 后的值。
  • 后缀形式：变量--
    • 先返回变量的当前值，然后对变量减 1。

示例

int a = 5;

// 前缀形式
int b = --a;  // a = 4, b = 4

// 后缀形式
int c = a--;  // a = 3, c = 4

3. 综合示例

以下是一个完整的C++程序，演示了自增和自减运算符的用法：

#include <iostream>
using namespace std;

int main() {
    int a = 10;

    // 自增运算符
    cout << "Initial value of a: " << a << endl;  // 输出：10

    int b = ++a;  // 前缀形式
    cout << "After prefix increment (++a): a = " << a << ", b = " << b << endl;  // 输出：a = 11, b = 11

    int c = a++;  // 后缀形式
    cout << "After postfix increment (a++): a = " << a << ", c = " << c << endl;  // 输出：a = 12, c = 11

    // 自减运算符
    int d = --a;  // 前缀形式
    cout << "After prefix decrement (--a): a = " << a << ", d = " << d << endl;  // 输出：a = 11, d = 11

    int e = a--;  // 后缀形式
    cout << "After postfix decrement (a--): a = " << a << ", e = " << e << endl;  // 输出：a = 10, e = 11

    return 0;
}

4. 注意事项

  1. 前缀与后缀的区别：

    • 前缀形式（++a 或 --a）先修改值，再使用变量。
    • 后缀形式（a++ 或 a--）先使用变量，再修改值。

  2. 副作用：

    • 自增和自减运算符会修改变量的值，因此在使用时要注意副作用。
    • 示例： 
      int a = 5;
int b = a++ + ++a;  // 行为未定义，避免在同一表达式中多次修改同一变量

  3. 非整数类型：

    • 自增和自减运算符也可以用于指针和迭代器。
    • 示例： 
      int arr[] = {1, 2, 3};
int* p = arr;
cout << *p++ << endl;  // 输出：1（先取 *p，然后 p++）
cout << *p << endl;    // 输出：2

5. 总结

++ 后缀形式 先使用值，再加 1 a++ a = 7, c = 6

-- 后缀形式 先使用值，再减 1 a-- a = 3, c = 4

自增和自减运算符是C++编程中常用的工具，熟练掌握它们可以编写出简洁高效的代码。
