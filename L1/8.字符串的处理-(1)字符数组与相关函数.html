在C++中，字符串可以通过字符数组或**std::string 类**来处理。字符数组是C风格的字符串表示方式，而 std::string 是C++标准库提供的更高级的字符串类。以下是字符数组及其相关函数的详细说明和示例：

1. 字符数组

字符数组是C风格字符串的基础，它以 \0（空字符）作为字符串的结束标志。

(1) 定义与初始化

  • 语法：

    char 数组名[大小];

  • 示例：

    char str1[10];  // 定义一个长度为 10 的字符数组
char str2[] = "Hello";  // 定义并初始化字符数组

  • 注意事项：

    • 字符数组的大小应足够容纳字符串及其结束符 \0。
    • 初始化时可以省略大小，编译器会自动计算。

(2) 访问元素

  • 通过下标访问字符数组中的元素。
  • 示例： 
    char str[] = "Hello";
cout << str[0] << endl;  // 输出：H
cout << str[4] << endl;  // 输出：o

(3) 遍历字符数组

  • 使用循环遍历字符数组，直到遇到 \0。
  • 示例： 
    char str[] = "Hello";
for (int i = 0; str[i] != '\0'; i++) {
    cout << str[i] << " ";
}
// 输出：H e l l o

2. 字符数组的相关函数

C++标准库 <cstring> 提供了许多用于操作字符数组的函数。

(1) strlen()

  • 作用：返回字符串的长度（不包括 \0）。
  • 语法： 
    size_t strlen(const char* str);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str[] = "Hello";
    cout << "Length: " << strlen(str) << endl;  // 输出：5
    return 0;
}

(2) strcpy()

  • 作用：将一个字符串复制到另一个字符串。
  • 语法： 
    char* strcpy(char* dest, const char* src);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char src[] = "Hello";
    char dest[10];
    strcpy(dest, src);
    cout << "Copied string: " << dest << endl;  // 输出：Hello
    return 0;
}

(3) strcat()

  • 作用：将一个字符串追加到另一个字符串的末尾。
  • 语法： 
    char* strcat(char* dest, const char* src);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str1[20] = "Hello";
    char str2[] = " World";
    strcat(str1, str2);
    cout << "Concatenated string: " << str1 << endl;  // 输出：Hello World
    return 0;
}

(4) strcmp()

  • 作用：比较两个字符串。
    • 返回值为 0：字符串相等。
    • 返回值小于 0：第一个字符串小于第二个字符串。
    • 返回值大于 0：第一个字符串大于第二个字符串。
  • 语法： 
    int strcmp(const char* str1, const char* str2);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str1[] = "Hello";
    char str2[] = "World";
    int result = strcmp(str1, str2);
    if (result == 0) {
        cout << "Strings are equal" << endl;
    } else if (result < 0) {
        cout << "str1 is less than str2" << endl;  // 输出
    } else {
        cout << "str1 is greater than str2" << endl;
    }
    return 0;
}

(5) strchr()

  • 作用：在字符串中查找指定字符的第一次出现位置。
  • 语法： 
    char* strchr(const char* str, int ch);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str[] = "Hello";
    char* p = strchr(str, 'l');
    if (p != nullptr) {
        cout << "Found at position: " << (p - str) << endl;  // 输出：2
    } else {
        cout << "Character not found" << endl;
    }
    return 0;
}

(6) strstr()

  • 作用：在字符串中查找子字符串的第一次出现位置。
  • 语法： 
    char* strstr(const char* str, const char* substr);
  • 示例： 
    #include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str[] = "Hello World";
    char* p = strstr(str, "World");
    if (p != nullptr) {
        cout << "Found at position: " << (p - str) << endl;  // 输出：6
    } else {
        cout << "Substring not found" << endl;
    }
    return 0;
}

3. 综合示例

以下是一个完整的C++程序，演示了字符数组的定义、初始化和相关函数的使用：

#include <cstring>
#include <iostream>
using namespace std;

int main() {
    // 定义并初始化字符数组
    char str1[] = "Hello";
    char str2[20];

    // 复制字符串
    strcpy(str2, str1);
    cout << "Copied string: " << str2 << endl;  // 输出：Hello

    // 连接字符串
    strcat(str2, " World");
    cout << "Concatenated string: " << str2 << endl;  // 输出：Hello World

    // 比较字符串
    int result = strcmp(str1, str2);
    if (result == 0) {
        cout << "Strings are equal" << endl;
    } else if (result < 0) {
        cout << "str1 is less than str2" << endl;  // 输出
    } else {
        cout << "str1 is greater than str2" << endl;
    }

    // 查找字符
    char* p = strchr(str1, 'l');
    if (p != nullptr) {
        cout << "Found 'l' at position: " << (p - str1) << endl;  // 输出：2
    } else {
        cout << "Character not found" << endl;
    }

    // 查找子字符串
    p = strstr(str2, "World");
    if (p != nullptr) {
        cout << "Found 'World' at position: " << (p - str2) << endl;  // 输出：6
    } else {
        cout << "Substring not found" << endl;
    }

    return 0;
}

4. 总结

函数 作用 示例
strlen() 返回字符串的长度 strlen("Hello") 返回 5
strcpy() 复制字符串 strcpy(dest, src)
strcat() 连接字符串 strcat(str1, str2)
strcmp() 比较字符串 strcmp(str1, str2)
strchr() 查找字符 strchr(str, 'l')
strstr() 查找子字符串 strstr(str, "World")

字符数组是C++中处理字符串的基础工具，熟练掌握其用法可以高效地操作字符串。
