在C++中，条件语句用于根据条件执行不同的代码块。常见的条件语句包括 if 语句、switch 语句以及多层条件语句。以下是它们的详细说明和示例：

1. if 语句

if 语句用于根据条件判断是否执行某段代码。

语法

if (条件) {
    // 条件为真时执行的代码
}

示例

int age = 18;
if (age >= 18) {
    cout << "You are an adult." << endl;
}

2. if-else 语句

if-else 语句用于在条件为真时执行一个代码块，否则执行另一个代码块。

语法

if (条件) {
    // 条件为真时执行的代码
} else {
    // 条件为假时执行的代码
}

示例

int age = 16;
if (age >= 18) {
    cout << "You are an adult." << endl;
} else {
    cout << "You are a minor." << endl;
}

3. if-else if-else 语句

if-else if-else 语句用于处理多个条件。

语法

if (条件1) {
    // 条件1为真时执行的代码
} else if (条件2) {
    // 条件2为真时执行的代码
} else {
    // 所有条件都为假时执行的代码
}

示例

int score = 85;
if (score >= 90) {
    cout << "Grade: A" << endl;
} else if (score >= 80) {
    cout << "Grade: B" << endl;
} else if (score >= 70) {
    cout << "Grade: C" << endl;
} else {
    cout << "Grade: F" << endl;
}

4. switch 语句

switch 语句用于根据变量的值执行不同的代码块。

语法

switch (变量) {
    case 值1:
        // 变量等于值1时执行的代码
        break;
    case 值2:
        // 变量等于值2时执行的代码
        break;
    default:
        // 变量不等于任何 case 值时执行的代码
}

示例

int day = 3;
switch (day) {
    case 1:
        cout << "Monday" << endl;
        break;
    case 2:
        cout << "Tuesday" << endl;
        break;
    case 3:
        cout << "Wednesday" << endl;
        break;
    default:
        cout << "Invalid day" << endl;
}

5. 多层条件语句

多层条件语句是指嵌套的 if 或 switch 语句，用于处理更复杂的逻辑。

示例 1：嵌套 if 语句

int age = 20;
bool isStudent = true;

if (age >= 18) {
    if (isStudent) {
        cout << "You are an adult student." << endl;
    } else {
        cout << "You are an adult." << endl;
    }
} else {
    cout << "You are a minor." << endl;
}

示例 2：switch 嵌套 if 语句

int day = 3;
bool isHoliday = false;

switch (day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        if (isHoliday) {
            cout << "It's a holiday!" << endl;
        } else {
            cout << "It's a weekday." << endl;
        }
        break;
    case 6:
    case 7:
        cout << "It's a weekend." << endl;
        break;
    default:
        cout << "Invalid day" << endl;
}

总结

语句类型 作用 示例
if 根据条件执行代码 if (age >= 18) { ... }
if-else 根据条件执行不同代码块 if (age >= 18) { ... } else { ... }
if-else if-else 处理多个条件 if (score >= 90) { ... } else if ...
switch 根据变量值执行不同代码块 switch (day) { case 1: ... }
多层条件语句 嵌套的条件语句，处理复杂逻辑 嵌套 if 或 switch

这些条件语句是C++编程中控制程序逻辑的核心工具，熟练掌握它们可以编写出更灵活和强大的程序。
