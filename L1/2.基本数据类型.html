在C++中，基本数据类型是语言内置的数据类型，用于定义变量和存储数据。C++的基本数据类型可以分为以下几类：

1. 整数类型

用于存储整数值，包括正数、负数和零。

数据类型 大小（字节） 范围
int 4 -2,147,483,648 到 2,147,483,647
short 2 -32,768 到 32,767
long 4 或 8 -2,147,483,648 到 2,147,483,647（4字节）
long long 8 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807
unsigned int 4 0 到 4,294,967,295
unsigned short 2 0 到 65,535
unsigned long 4 或 8 0 到 4,294,967,295（4字节）
unsigned long long 8 0 到 18,446,744,073,709,551,615

示例

int age = 25;
short count = 100;
long population = 8000000000L;
unsigned int distance = 3000;

2. 浮点类型

用于存储小数值（实数）。

数据类型 大小（字节） 范围（近似） 精度
float 4 ±3.4e-38 到 ±3.4e38 6-7 位小数
double 8 ±1.7e-308 到 ±1.7e308 15-16 位小数
long double 8 或 16 ±1.7e-308 到 ±1.7e308（扩展精度） 19-20 位小数

示例

float pi = 3.14f;
double gravity = 9.81;
long double atomicMass = 1.66053906660e-27L;

3. 字符类型

用于存储单个字符。

数据类型 大小（字节） 范围
char 1 -128 到 127 或 0 到 255
unsigned char 1 0 到 255
wchar_t 2 或 4 宽字符（支持 Unicode）

示例

char grade = 'A';
unsigned char byte = 255;
wchar_t wideChar = L'中';

4. 布尔类型

用于存储逻辑值，只有两个可能的值：true 或 false。

数据类型 大小（字节） 值
bool 1 true 或 false

示例

bool isCppFun = true;
bool isRaining = false;

5. 空类型

用于表示无类型或空值。

数据类型 描述
void 表示无类型，通常用于函数返回值或指针

示例

void printMessage() {
    cout << "Hello, World!" << endl;
}

6. 修饰符

C++ 提供了类型修饰符，用于改变基本数据类型的含义：

  • signed：表示有符号数（默认）。
  • unsigned：表示无符号数（仅非负数）。
  • short：减小数据类型的大小。
  • long：增大数据类型的大小。

示例

unsigned int positiveNumber = 100;
long double highPrecisionValue = 3.141592653589793238L;

总结

C++ 的基本数据类型包括：

  1. 整数类型：int, short, long, long long 等。
  2. 浮点类型：float, double, long double。
  3. 字符类型：char, unsigned char, wchar_t。
  4. 布尔类型：bool。
  5. 空类型：void。

这些数据类型是C++编程的基础，开发者可以根据需要选择合适的数据类型来存储和操作数据。
