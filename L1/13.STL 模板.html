在C++中，STL（Standard Template Library，标准模板库） 提供了丰富的模板类和函数，用于简化常见的数据结构和算法操作。以下是STL中常用的函数和容器的详细说明和示例：

1. 常用算法模板函数

STL中的 <algorithm> 头文件提供了许多常用的算法函数，如 min、max、swap 和 sort。

(1) min 和 max

  • 作用：返回两个值中的最小值或最大值。
  • 语法： 
    template <class T>
const T& min(const T& a, const T& b);

template <class T>
const T& max(const T& a, const T& b);
  • 示例： 
    #include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a = 10, b = 20;
    cout << "Min: " << min(a, b) << endl;  // 输出：10
    cout << "Max: " << max(a, b) << endl;  // 输出：20
    return 0;
}

(2) swap

  • 作用：交换两个值。
  • 语法： 
    template <class T>
void swap(T& a, T& b);
  • 示例： 
    #include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a = 10, b = 20;
    swap(a, b);
    cout << "a: " << a << ", b: " << b << endl;  // 输出：a: 20, b: 10
    return 0;
}

(3) sort

  • 作用：对容器中的元素进行排序。
  • 语法： 
    template <class RandomAccessIterator>
void sort(RandomAccessIterator first, RandomAccessIterator last);
  • 示例： 
    #include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main() {
    vector<int> vec = {5, 3, 1, 4, 2};
    sort(vec.begin(), vec.end());  // 对 vector 进行排序
    for (int num : vec) {
        cout << num << " ";  // 输出：1 2 3 4 5
    }
    cout << endl;
    return 0;
}

2. 常用容器

STL提供了多种容器，如 stack、queue 等，用于存储和管理数据。

(1) 栈（stack）

  • 特点：后进先出（LIFO）。
  • 常用操作：
    • push()：将元素压入栈。
    • pop()：弹出栈顶元素。
    • top()：返回栈顶元素。
    • empty()：判断栈是否为空。
    • size()：返回栈中元素的数量。
  • 示例： 
    #include <iostream>
#include <stack>
using namespace std;

int main() {
    stack<int> s;
    s.push(10);  // 压入元素
    s.push(20);
    s.push(30);

    cout << "Top element: " << s.top() << endl;  // 输出：30

    s.pop();  // 弹出栈顶元素
    cout << "Top element after pop: " << s.top() << endl;  // 输出：20

    cout << "Stack size: " << s.size() << endl;  // 输出：2
    return 0;
}

(2) 队列（queue）

  • 特点：先进先出（FIFO）。
  • 常用操作：
    • push()：将元素加入队列。
    • pop()：移除队首元素。
    • front()：返回队首元素。
    • back()：返回队尾元素。
    • empty()：判断队列是否为空。
    • size()：返回队列中元素的数量。
  • 示例： 
    #include <iostream>
#include <queue>
using namespace std;

int main() {
    queue<int> q;
    q.push(10);  // 加入元素
    q.push(20);
    q.push(30);

    cout << "Front element: " << q.front() << endl;  // 输出：10
    cout << "Back element: " << q.back() << endl;    // 输出：30

    q.pop();  // 移除队首元素
    cout << "Front element after pop: " << q.front() << endl;  // 输出：20

    cout << "Queue size: " << q.size() << endl;  // 输出：2
    return 0;
}

3. 综合示例

以下是一个完整的C++程序，演示了STL中的算法函数和容器的使用：

#include <iostream>
#include <algorithm>
#include <vector>
#include <stack>
#include <queue>
using namespace std;

int main() {
    // 使用 min 和 max
    int a = 10, b = 20;
    cout << "Min: " << min(a, b) << endl;  // 输出：10
    cout << "Max: " << max(a, b) << endl;  // 输出：20

    // 使用 swap
    swap(a, b);
    cout << "After swap: a = " << a << ", b = " << b << endl;  // 输出：a = 20, b = 10

    // 使用 sort
    vector<int> vec = {5, 3, 1, 4, 2};
    sort(vec.begin(), vec.end());
    cout << "Sorted vector: ";
    for (int num : vec) {
        cout << num << " ";  // 输出：1 2 3 4 5
    }
    cout << endl;

    // 使用 stack
    stack<int> s;
    s.push(10);
    s.push(20);
    s.push(30);
    cout << "Stack top: " << s.top() << endl;  // 输出：30
    s.pop();
    cout << "Stack top after pop: " << s.top() << endl;  // 输出：20

    // 使用 queue
    queue<int> q;
    q.push(10);
    q.push(20);
    q.push(30);
    cout << "Queue front: " << q.front() << endl;  // 输出：10
    cout << "Queue back: " << q.back() << endl;    // 输出：30
    q.pop();
    cout << "Queue front after pop: " << q.front() << endl;  // 输出：20

    return 0;
}

4. 总结

函数/容器 说明 示例
min 返回两个值中的最小值 min(a, b)
max 返回两个值中的最大值 max(a, b)
swap 交换两个值 swap(a, b)
sort 对容器中的元素进行排序 sort(vec.begin(), vec.end())
stack 后进先出（LIFO）的容器 stack s; s.push(10); s.pop();
queue 先进先出（FIFO）的容器 queue q; q.push(10); q.pop();

STL是C++编程中强大的工具库，熟练掌握其用法可以极大地提高编程效率。
