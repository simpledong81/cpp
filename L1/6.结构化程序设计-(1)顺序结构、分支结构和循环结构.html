在C++中，结构化程序设计是一种编程范式，强调使用清晰、模块化的结构来组织代码。结构化程序设计的核心包括以下三种基本控制结构：

  1. 顺序结构：代码按顺序逐行执行。
  2. 分支结构：根据条件选择执行不同的代码块。
  3. 循环结构：重复执行某段代码，直到满足特定条件。

以下是这三种结构的详细说明和示例：

1. 顺序结构

顺序结构是程序的基本执行方式，代码从上到下逐行执行。

示例

#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 20;
    int sum = a + b;

    cout << "Sum: " << sum << endl;  // 输出：Sum: 30
    return 0;
}

2. 分支结构

分支结构根据条件选择执行不同的代码块。C++中常用的分支结构包括 if、if-else 和 switch。

(1) if 语句

  • 作用：如果条件为真，则执行代码块。
  • 语法： 
    if (条件) {
    // 条件为真时执行的代码
}
  • 示例： 
    int age = 18;
if (age >= 18) {
    cout << "You are an adult." << endl;
}

(2) if-else 语句

  • 作用：如果条件为真，执行第一个代码块；否则执行第二个代码块。
  • 语法： 
    if (条件) {
    // 条件为真时执行的代码
} else {
    // 条件为假时执行的代码
}
  • 示例： 
    int age = 16;
if (age >= 18) {
    cout << "You are an adult." << endl;
} else {
    cout << "You are a minor." << endl;
}

(3) if-else if-else 语句

  • 作用：处理多个条件。
  • 语法： 
    if (条件1) {
    // 条件1为真时执行的代码
} else if (条件2) {
    // 条件2为真时执行的代码
} else {
    // 所有条件都为假时执行的代码
}
  • 示例： 
    int score = 85;
if (score >= 90) {
    cout << "Grade: A" << endl;
} else if (score >= 80) {
    cout << "Grade: B" << endl;
} else if (score >= 70) {
    cout << "Grade: C" << endl;
} else {
    cout << "Grade: F" << endl;
}

(4) switch 语句

  • 作用：根据变量的值选择执行不同的代码块。
  • 语法： 
    switch (变量) {
    case 值1:
        // 变量等于值1时执行的代码
        break;
    case 值2:
        // 变量等于值2时执行的代码
        break;
    default:
        // 变量不等于任何 case 值时执行的代码
}
  • 示例： 
    int day = 3;
switch (day) {
    case 1:
        cout << "Monday" << endl;
        break;
    case 2:
        cout << "Tuesday" << endl;
        break;
    case 3:
        cout << "Wednesday" << endl;
        break;
    default:
        cout << "Invalid day" << endl;
}

3. 循环结构

循环结构用于重复执行某段代码，直到满足特定条件。C++中常用的循环结构包括 for、while 和 do-while。

(1) for 循环

  • 作用：在已知循环次数的情况下重复执行代码。
  • 语法： 
    for (初始化; 条件; 更新) {
    // 循环体
}
  • 示例： 
    for (int i = 0; i < 5; i++) {
    cout << "i = " << i << endl;
}

(2) while 循环

  • 作用：在条件为真时重复执行代码。
  • 语法： 
    while (条件) {
    // 循环体
}
  • 示例： 
    int i = 0;
while (i < 5) {
    cout << "i = " << i << endl;
    i++;
}

(3) do-while 循环

  • 作用：先执行一次代码，然后在条件为真时重复执行。
  • 语法： 
    do {
    // 循环体
} while (条件);
  • 示例： 
    int i = 0;
do {
    cout << "i = " << i << endl;
    i++;
} while (i < 5);

4. 综合示例

以下是一个完整的C++程序，演示了顺序结构、分支结构和循环结构的结合使用：

#include <iostream>
using namespace std;

int main() {
    // 顺序结构
    int a = 10, b = 20;
    int sum = a + b;
    cout << "Sum: " << sum << endl;

    // 分支结构
    int age = 16;
    if (age >= 18) {
        cout << "You are an adult." << endl;
    } else {
        cout << "You are a minor." << endl;
    }

    // 循环结构
    for (int i = 0; i < 5; i++) {
        cout << "i = " << i << endl;
    }

    return 0;
}

5. 总结

结构类型 特点 适用场景 示例
顺序结构 代码按顺序逐行执行 基本程序逻辑 int sum = a + b;
分支结构 根据条件选择执行不同的代码块 条件判断 if (age >= 18) { ... } else { ... }
循环结构 重复执行某段代码，直到满足特定条件 遍历数组、重复操作 for (int i = 0; i < 5; i++) { ... }

结构化程序设计是C++编程的基础，掌握这些控制结构可以编写出清晰、高效的代码。
