在C++中，算术运算是最基本的数学运算，用于对数值类型的数据进行计算。以下是C++中常见的算术运算符及其用法：

1. 加法运算符 (+)

  • 作用：计算两个操作数的和。
  • 示例： 
    int a = 10, b = 20;
int sum = a + b;  // sum = 30

2. 减法运算符 (-)

  • 作用：计算两个操作数的差。
  • 示例： 
    int a = 20, b = 10;
int diff = a - b;  // diff = 10

3. 乘法运算符 (*)

  • 作用：计算两个操作数的积。
  • 示例： 
    int a = 5, b = 6;
int product = a * b;  // product = 30

4. 除法运算符 (/)

  • 作用：计算两个操作数的商。
    • 如果操作数都是整数，结果为整数（整除）。
    • 如果操作数中有浮点数，结果为浮点数。
  • 示例： 
    int a = 10, b = 3;
int quotient1 = a / b;  // quotient1 = 3（整除）

double c = 10.0, d = 3.0;
double quotient2 = c / d;  // quotient2 = 3.33333

5. 整除运算符 (/)

  • 作用：当操作数都是整数时，/ 执行整除运算，结果舍弃小数部分。
  • 示例： 
    int a = 10, b = 3;
int quotient = a / b;  // quotient = 3

6. 求余运算符 (%)

  • 作用：计算两个整数相除的余数。
    • 只能用于整数类型。
  • 示例： 
    int a = 10, b = 3;
int remainder = a % b;  // remainder = 1

7. 综合示例

以下是一个完整的C++程序，演示了各种算术运算的用法：

#include <iostream>
using namespace std;

int main() {
    int a = 20, b = 6;

    // 加法
    int sum = a + b;
    cout << "Sum: " << sum << endl;  // 输出：Sum: 26

    // 减法
    int diff = a - b;
    cout << "Difference: " << diff << endl;  // 输出：Difference: 14

    // 乘法
    int product = a * b;
    cout << "Product: " << product << endl;  // 输出：Product: 120

    // 除法（整除）
    int quotient1 = a / b;
    cout << "Quotient (int): " << quotient1 << endl;  // 输出：Quotient (int): 3

    // 除法（浮点数）
    double quotient2 = static_cast<double>(a) / b;
    cout << "Quotient (double): " << quotient2 << endl;  // 输出：Quotient (double): 3.33333

    // 求余
    int remainder = a % b;
    cout << "Remainder: " << remainder << endl;  // 输出：Remainder: 2

    return 0;
}

8. 注意事项

  1. 整数除法：如果两个整数相除，结果会被截断为整数。 
    int a = 5, b = 2;
int result = a / b;  // result = 2
  2. 浮点数除法：如果至少有一个操作数是浮点数，结果会是浮点数。 
    double a = 5.0, b = 2;
double result = a / b;  // result = 2.5
  3. 求余运算：只能用于整数类型，不能用于浮点数。 
    int a = 10, b = 3;
int remainder = a % b;  // remainder = 1

总结

运算符 作用 示例 结果（示例）
+ 加法 10 + 20 30
- 减法 20 - 10 10
* 乘法 5 * 6 30
/ 除法 10 / 3（整数） 3
/ 浮点除法 10.0 / 3 3.33333
% 求余 10 % 3 1

这些算术运算符是C++编程中最基础的工具，熟练掌握它们可以轻松实现各种数学计算。
