在C++中，数组的读入和输出是常见的操作。通过循环和标准输入输出流（cin 和 cout），可以方便地实现数组的读入和输出。以下是详细说明和示例：

1. 数组的读入

数组的读入通常使用 cin 结合循环来实现。

示例：读入一维数组

#include <iostream>
using namespace std;

int main() {
    const int size = 5;  // 数组大小
    int numbers[size];   // 定义数组

    // 读入数组
    cout << "Enter " << size << " numbers: ";
    for (int i = 0; i < size; i++) {
        cin >> numbers[i];
    }

    // 输出数组
    cout << "You entered: ";
    for (int i = 0; i < size; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;

    return 0;
}

示例：读入二维数组

#include <iostream>
using namespace std;

int main() {
    const int rows = 2, cols = 3;  // 数组大小
    int matrix[rows][cols];         // 定义二维数组

    // 读入二维数组
    cout << "Enter " << rows << "x" << cols << " matrix:" << endl;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            cin >> matrix[i][j];
        }
    }

    // 输出二维数组
    cout << "The matrix is:" << endl;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

2. 数组的输出

数组的输出通常使用 cout 结合循环来实现。

示例：输出一维数组

#include <iostream>
using namespace std;

int main() {
    const int size = 5;  // 数组大小
    int numbers[size] = {1, 2, 3, 4, 5};  // 初始化数组

    // 输出数组
    cout << "Array elements: ";
    for (int i = 0; i < size; i++) {
        cout << numbers[i] << " ";
    }
    cout << endl;

    return 0;
}

示例：输出二维数组

#include <iostream>
using namespace std;

int main() {
    const int rows = 2, cols = 3;  // 数组大小
    int matrix[rows][cols] = {
        {1, 2, 3},
        {4, 5, 6}
    };  // 初始化二维数组

    // 输出二维数组
    cout << "The matrix is:" << endl;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }

    return 0;
}

4. 注意事项

  1. 数组大小：

    • 数组的大小在定义时必须确定，且不能动态改变。
    • 可以使用常量或宏定义数组大小，便于维护。

  2. 下标越界：

    • 访问数组时，下标必须在 0 到 数组大小 - 1 的范围内，否则会导致未定义行为。

  3. 输入格式：

    • 输入时，数据之间可以用空格或换行分隔。

  4. 输出格式：

    • 输出时，可以使用空格、换行或制表符（\t）格式化输出。
