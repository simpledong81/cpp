在C++中，函数是程序的基本模块，用于封装可重用的代码逻辑。函数通过定义和调用来实现其功能。函数的参数分为形参（形式参数）和实参（实际参数）。以下是函数定义与调用、形参与实参的详细说明和示例：

1. 函数的定义

函数的定义包括函数名、返回类型、参数列表和函数体。

语法

返回类型 函数名(参数列表) {
    // 函数体
    return 返回值;  // 如果返回类型不是 void
}

示例

#include <iostream>
using namespace std;

// 定义一个函数，计算两个整数的和
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5);  // 调用函数
    cout << "Sum: " << result << endl;  // 输出：Sum: 8
    return 0;
}

2. 函数的调用

函数通过函数名和实参列表来调用。

语法

函数名(实参列表);

示例

int sum = add(10, 20);  // 调用 add 函数，实参为 10 和 20

3. 形参与实参

  • 形参：函数定义时声明的参数，用于接收调用时传递的值。
  • 实参：函数调用时传递的具体值。

示例

#include <iostream>
using namespace std;

// 定义一个函数，形参为 a 和 b
void printSum(int a, int b) {
    cout << "Sum: " << (a + b) << endl;
}

int main() {
    int x = 5, y = 10;
    printSum(x, y);  // 调用函数，实参为 x 和 y
    return 0;
}

4. 函数的返回类型

  • 函数可以返回一个值，返回类型由函数定义时指定。
  • 如果函数不需要返回值，返回类型应为 void。

示例

#include <iostream>
using namespace std;

// 返回类型为 int
int multiply(int a, int b) {
    return a * b;
}

// 返回类型为 void
void printMessage() {
    cout << "Hello, World!" << endl;
}

int main() {
    int result = multiply(3, 4);  // 调用 multiply 函数
    cout << "Product: " << result << endl;  // 输出：Product: 12

    printMessage();  // 调用 printMessage 函数
    return 0;
}

5. 函数的默认参数

C++允许为函数的参数提供默认值。如果调用时未传递该参数，则使用默认值。

语法

返回类型 函数名(参数类型 参数名 = 默认值);

示例

#include <iostream>
using namespace std;

// 默认参数 b = 10
int add(int a, int b = 10) {
    return a + b;
}

int main() {
    cout << add(5) << endl;    // 输出：15（使用默认值 b = 10）
    cout << add(5, 20) << endl; // 输出：25（传递 b = 20）
    return 0;
}

6. 函数的重载

C++支持函数重载，即定义多个同名函数，但参数列表不同（参数类型或数量不同）。

示例

#include <iostream>
using namespace std;

// 重载 add 函数
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int main() {
    cout << add(3, 5) << endl;          // 输出：8（调用 int 版本）
    cout << add(3.5, 2.5) << endl;      // 输出：6.0（调用 double 版本）
    return 0;
}

7. 递归函数

递归函数是指在函数体内调用自身的函数。递归通常用于解决可以分解为相似子问题的问题。

示例：计算阶乘

#include <iostream>
using namespace std;

// 递归函数：计算阶乘
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);  // 递归调用
}

int main() {
    int n = 5;
    cout << "Factorial of " << n << " is: " << factorial(n) << endl;  // 输出：120
    return 0;
}

8. 综合示例

以下是一个完整的C++程序，演示了函数的定义、调用、形参与实参、默认参数、函数重载和递归：

#include <iostream>
using namespace std;

// 函数定义：计算两个整数的和
int add(int a, int b) {
    return a + b;
}

// 默认参数
int multiply(int a, int b = 10) {
    return a * b;
}

// 函数重载
double add(double a, double b) {
    return a + b;
}

// 递归函数：计算阶乘
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

int main() {
    // 调用 add 函数
    cout << "Sum (int): " << add(3, 5) << endl;          // 输出：8
    cout << "Sum (double): " << add(3.5, 2.5) << endl;  // 输出：6.0

    // 调用 multiply 函数
    cout << "Product (default): " << multiply(5) << endl;    // 输出：50
    cout << "Product: " << multiply(5, 20) << endl;          // 输出：100

    // 调用递归函数
    cout << "Factorial of 5: " << factorial(5) << endl;      // 输出：120

    return 0;
}

9. 总结

概念 说明 示例
函数定义 定义函数名、返回类型、参数列表和函数体 int add(int a, int b) { ... }
函数调用 通过函数名和实参列表调用函数 add(3, 5)
形参与实参 形参是函数定义中的参数，实参是调用时传递的值 void printSum(int a, int b)
默认参数 为函数参数提供默认值 int multiply(int a, int b = 10)
函数重载 定义多个同名函数，参数列表不同 int add(int a, int b) 和 double add(double a, double b)
递归函数 函数调用自身 int factorial(int n) { ... }

函数是C++编程中的核心概念，熟练掌握函数的定义、调用和参数传递可以编写出模块化、可重用的代码。
