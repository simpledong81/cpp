在C++中，常量和变量是程序中用于存储数据的基本元素。它们的命名、定义及作用如下：

1. 常量（Constant）

常量是指在程序运行期间其值不能被修改的量。

命名规则

  • 常量的命名通常使用大写字母，单词之间用下划线 _ 分隔，以增强可读性。
  • 示例：MAX_SIZE, PI, DEFAULT_TIMEOUT。

定义方式

  1. 使用 const 关键字

    • const 定义的常量在编译时确定其值，且不能修改。
    • 语法： 
      const 数据类型 常量名 = 值;
    • 示例： 
      const int MAX_SIZE = 100;
const double PI = 3.14159;

  2. 使用 #define 宏定义

    • #define 是预处理指令，用于定义常量，没有类型检查。
    • 语法： 
      #define 常量名 值
    • 示例： 
      #define PI 3.14159
#define MAX_SIZE 100

作用

  • 提高代码的可读性和可维护性。
  • 防止意外修改数据。
  • 用于存储程序中不变的值，如数学常数、配置参数等。

2. 变量（Variable）

变量是程序中用于存储数据的命名内存位置，其值可以在程序运行期间改变。

命名规则

  • 变量名由字母、数字和下划线组成，但不能以数字开头。
  • 变量名区分大小写。
  • 不能使用C++关键字作为变量名。
  • 命名应具有描述性，通常使用小写字母，单词之间用下划线 _ 分隔或驼峰命名法。
  • 示例：age, total_score, userName。

定义方式

  • 语法： 
    数据类型 变量名;          // 声明变量
数据类型 变量名 = 值;    // 声明并初始化变量
  • 示例： 
    int age;                // 声明一个整型变量
double price = 99.99;   // 声明并初始化一个双精度浮点型变量

作用

  • 存储程序运行期间需要修改的数据。
  • 提供一种灵活的方式来操作内存中的数据。
  • 用于临时存储计算结果、用户输入、循环计数器等。

常量与变量的区别

特性 常量（Constant） 变量（Variable）
值是否可变 不可变 可变
定义方式 const 或 #define 直接声明
命名习惯 通常全大写，单词间用下划线分隔 通常小写，单词间用下划线或驼峰命名
作用 存储程序中不变的值 存储程序中可变的值

示例代码

#include <iostream>
using namespace std;

#define MAX_SIZE 100  // 使用 #define 定义常量

int main() {
    const double PI = 3.14159;  // 使用 const 定义常量
    int radius = 5;             // 定义变量
    double area = PI * radius * radius;  // 使用常量和变量计算

    cout << "Area of the circle: " << area << endl;
    cout << "Max size: " << MAX_SIZE << endl;

    radius = 10;  // 修改变量的值
    area = PI * radius * radius;
    cout << "New area of the circle: " << area << endl;

    return 0;
}

总结

  • 常量用于存储程序中不变的值，定义后不能被修改，通常使用 const 或 #define。
  • 变量用于存储程序中可变的值，可以在程序运行期间修改其值。
  • 良好的命名习惯可以提高代码的可读性和可维护性。
