在C++中，程序的基本语句用于实现输入、输出、赋值和控制程序逻辑。以下是常见的几种基本语句及其用法：

1. cin 语句

  • 作用：用于从标准输入（通常是键盘）读取数据。
  • 语法： 
    cin >> 变量;
  • 示例： 
    int age;
cout << "Enter your age: ";
cin >> age;  // 从键盘读取一个整数并存储到 age 变量中

2. scanf 语句

  • 作用：C语言风格的输入函数，用于从标准输入读取格式化数据。
  • 语法： 
    scanf("格式控制字符串", &变量);
  • 示例： 
    int age;
printf("Enter your age: ");
scanf("%d", &age);  // 从键盘读取一个整数并存储到 age 变量中

3. cout 语句

  • 作用：用于向标准输出（通常是屏幕）打印数据。
  • 语法： 
    cout << 数据;
  • 示例： 
    int age = 25;
cout << "Your age is: " << age << endl;  // 输出变量 age 的值

4. printf 语句

  • 作用：C语言风格的输出函数，用于向标准输出打印格式化数据。
  • 语法： 
    printf("格式控制字符串", 变量);
  • 示例： 
    int age = 25;
printf("Your age is: %d\n", age);  // 输出变量 age 的值

5. 赋值语句

  • 作用：将一个值赋给变量。
  • 语法： 
    变量 = 值;
  • 示例： 
    int x;
x = 10;  // 将 10 赋值给变量 x

6. 复合语句

  • 作用：将多条语句组合在一起，形成一个代码块。复合语句用花括号 {} 包围。
  • 语法： 
    {
    语句1;
    语句2;
    ...
}
  • 示例： 
    {
    int x = 10;
    int y = 20;
    int sum = x + y;
    cout << "Sum: " << sum << endl;
}

综合示例

以下是一个完整的C++程序，演示了上述基本语句的使用：

#include <iostream>
using namespace std;

int main() {
    // 使用 cin 和 cout
    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "Your age is: " << age << endl;

    // 使用 scanf 和 printf
    int height;
    printf("Enter your height (in cm): ");
    scanf("%d", &height);
    printf("Your height is: %d cm\n", height);

    // 赋值语句
    int x = 10;
    int y = 20;
    int sum = x + y;

    // 复合语句
    {
        int product = x * y;
        cout << "Product: " << product << endl;
    }

    cout << "Sum: " << sum << endl;

    return 0;
}

总结

语句类型 作用 示例
cin 从标准输入读取数据 cin >> age;
scanf C语言风格的格式化输入 scanf("%d", &age);
cout 向标准输出打印数据 cout << "Age: " << age << endl;
printf C语言风格的格式化输出 printf("Age: %d\n", age);
赋值语句 将值赋给变量 x = 10;
复合语句 将多条语句组合成一个代码块 { int x = 10; cout << x << endl; }

这些基本语句是C++编程的基础，掌握它们对于编写和理解C++程序至关重要。
